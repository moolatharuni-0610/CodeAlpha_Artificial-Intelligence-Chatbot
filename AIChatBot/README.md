# AI ChatBot using Java

## Project Overview

AI ChatBot is a Java-based chatbot application designed to provide interactive communication between users and the system. The chatbot uses Natural Language Processing (NLP) techniques and rule-based logic to understand user inputs and generate appropriate responses. The application features a graphical user interface (GUI) built using Java Swing for real-time interaction.

## Features

* Interactive chatbot conversation
* User-friendly Java Swing GUI
* Rule-based Natural Language Processing (NLP)
* Answers frequently asked questions
* Date and time information
* Programming-related responses
* Help and support commands
* Real-time communication

## Technologies Used

* Java
* Java Swing
* OOP Concepts
* NLP (Keyword Matching)
* Event Handling

## Working Principle

1. The user enters a message in the chatbot interface.
2. The chatbot receives the input.
3. The input is converted to lowercase for processing.
4. NLP-based keyword matching identifies the user's intent.
5. The chatbot generates an appropriate response.
6. The response is displayed in the chat window.

## Supported Commands

| User Input  | ChatBot Response       |
| ----------- | ---------------------- |
| Hello / Hi  | Greeting message       |
| How are you | Status response        |
| Your name   | Bot introduction       |
| Date        | Current date           |
| Time        | Current time           |
| Java        | Information about Java |
| Project     | Project details        |
| Help        | Available commands     |
| Bye         | Exit message           |

## Installation

### Prerequisites

* Java JDK 8 or above
* Any Java IDE (Eclipse, IntelliJ IDEA, NetBeans, VS Code)

### Steps

1. Download or clone the project.
2. Open the project in your preferred IDE.
3. Compile the Java files.

```
javac ChatBotLogic.java
javac ChatBotGUI.java
```

4. Run the application.

```
java ChatBotGUI
```
